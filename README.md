# game-night-project
This project is part of the Google Student Ambassador Program, where we explore Google tools through hands-on learning. We created an interactive Snake and Ladder game using Canvas, turning simple ideas into fun and creative applications while improving our coding and design skills.

# 🎲 Roll & Rise
Roll & Rise is a fun and interactive board game inspired by the classic Snake and Ladder. Built using Canvas, this game allows players to roll a dice, move across the board, climb ladders to rise faster, and avoid snakes that bring them down. The main objective is to reach the final position before others, making the game both exciting and engaging while showcasing basic game logic and creativity.
